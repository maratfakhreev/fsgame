# ReactNativeBase
Kick-start your new mobile application based on React Native and Reflux technologies.

## NPM Dependencies:
    "react"
    "react-native"
    "react-native-animatable"
    "react-native-blur"
    "react-native-vector-icons"
    "reflux"
    "socket.io-client"
    "tcomb"
    "tcomb-form-native"
    "tcomb-validation"

##Start

```shell
npm start
```
